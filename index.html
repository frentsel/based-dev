Алгоритм

Это последовательность шагов (команд,инструкций)
для достижения конечной цели

Основные виды алгоритмов:

  1. Линейный;
  2. Циклический;
  3. Ветвление;
  4. Смешанный.

Основные свойства алгоритмов:

  1. Дискретность - алгоритм состоит из шагов (команд, инструкций);
  2. Конечность - алгоритм должен вести к конечному результату;
  3. Определенность - каждая команда должна точно и однозначно определять действие;
  4. Понятность - исполняемые команды должны быть понятны исполнителю.
